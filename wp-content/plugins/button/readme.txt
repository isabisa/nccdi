== Installation ==
In WordPress:

1. Go to `Plugins` > `Add New` > search for `button`
1. Press `Install Now` for the Button plugin
1. Press `Activate Plugin`

Manual installation:

1. Upload the `button` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the `Plugins` menu in WordPress
